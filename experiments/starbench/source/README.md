# BKR/REF source dump

The full source dump is intentionally not tracked: it is approximately 361 MB
and is third-party data. Download version 0.2 from the
[REF Zenodo record](https://doi.org/10.5281/zenodo.4148888) and place it here as:

```text
experiments/starbench/source/BKR-star-fullKGdump.ttls.gz
```

Expected MD5:

```text
eb91d948b94c3d7607c4f07650433e16
```

`experiments/generate_starbench_samples.py` verifies this checksum before
regenerating the checked-in RDF 1.2 samples.
